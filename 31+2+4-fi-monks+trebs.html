31+2+4 FI - Monks + Trebs

Good strategy for flanks in team games for civs with a strong monastery. Below build is optimised for the scenario requirements which is very gold-heavy, but you will want more on wood/farms if adding pikes for example. See link below for an in-depth guide - this uses a different build with -1 pop in Dark age, but the same Imperial time. This build requires more market abuse, but may be preferable if you end up needing the castle up sooner to defend.

The scenario for these builds uses Magyars because they are completely generic here... but they are not exactly recommended for this strategy.
